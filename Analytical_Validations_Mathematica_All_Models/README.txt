Mathematica_AllModels.nb: The mathematica notebook validates expressions derived in each of the model discussed in the manuscript.

*.pdf files: enable viewing above mathematica notebook in adobe viewer without installing Woldram Mathematica software. The smaller file simply lists the index identifying locations of the each model and the larger file contains the index and the actual calculations themselves.

The Mathematica notebook was written in version 12.0.0 
Mathematica software can be puchased and downloaded from the link below.
https://www.wolfram.com/mathematica/
